# GroceryHub - Online Grocery Store

GroceryHub is a simple web application built using **Python Flask** that allows users to browse grocery products, add items to a cart, and create an account.

## Features

- Home page with promotions and products
- Shop page to view grocery items
- Add items to cart
- Cart page with total price calculation
- Login system
- Register (create account) page
- Responsive UI with HTML, CSS, and JavaScript

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
